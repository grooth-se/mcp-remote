Import Certificates from Excel

Expected Excel Format

The Excel file should have the following columns in the header row:

  • Year - Certificate year (e.g., 2026)
  • ID - Certificate ID number (e.g., 1001)
  • Revision No - Revision number (default: 1)
  • Date - Certificate date
  • Test Project, Project name, Test standard
  • Customer, Customer order
  • Product, Product S/N, Material/ HT
  • Specimen ID, Location/orient.:, Temperature
  • Comment, Reported, Invoiced
